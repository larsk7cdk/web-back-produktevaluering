# Produkt evaluering

### Opgave
I denne opgave skal du lave en web-applikation, der kan bruges til at evaluere et produkt. Brugeren skal kunne indtaste sit navn, navnet på et produkt og så give produktet en karakter fra 1-10.

Du skal lave projektet som en .NET Core 2.0 projekt - ligesom kryds og bolle-spillet i kapitel 4.

- Du behøver ikke at bruge Middleware
- Du behøver ikke at bruge bootstrap og andre tredjeparts biblioteker fra Bower
